# DaviRain Log

Personal technical blog built with Next.js + Tailwind CSS.

![Design](https://img.shields.io/badge/Design-Ship%20List-black)
![Tech](https://img.shields.io/badge/Tech-Next.js%20+%20Tailwind-black)

## ✨ Features

- **Ship List Design**: Timeline-based article layout
- **Dark Theme**: Pure black `#030303` with subtle gradients
- **Typography**: Inter + JetBrains Mono
- **Responsive**: Mobile-friendly grid layout

## 🚀 Deploy

```bash
npm install
npm run build
```

Deploy `dist/` folder to Vercel or GitHub Pages.

## 📝 Migration from Jekyll

This blog was originally built with Jekyll and has been migrated to Next.js.
All existing posts have been preserved in the timeline layout.
